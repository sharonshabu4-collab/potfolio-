# Sharon Shabu - Personal Portfolio

A modern, responsive, and beautifully designed personal portfolio website showcasing my skills in Electrical & Electronics Engineering, 3D modeling, and my passion for software development.

## 🚀 Key Features

- **Modern Dark Theme UI:** Slick, high-contrast user interface with neon gradient aesthetics and smooth glassmorphism.
- **Scroll Animations:** Smooth Intersection Observer-based enter animations as you scroll down the page.
- **Interactive Mini-Bot:** A custom floating chatbot that can automatically reply to common visitor questions.
- **Certificate Modal:** A click-to-preview image modal allowing visitors to verify certificates natively.
- **Fully Responsive:** Adapts seamlessly on desktop, tablet, and mobile viewing environments.

## 🛠️ Tech Stack

- **HTML5:** Semantic architecture.
- **Vanilla CSS3:** Custom variables, grid/flexbox layouts, responsive media queries, and specialized keyframe animations. (No bloated CSS frameworks).
- **Vanilla JavaScript:** Used heavily for the scrolling engine, interaction modals, and logical chatbot scripts.

## 📁 File Structure

```text
├── index.html     - Main semantic structure of the one-page portfolio
├── styles.css     - Contains all design systems, themes, and animations
├── script.js      - All client-side javascript logic for interactions
└── assets/
    └── avatar.png - The 3D avatar graphic
```

## 💻 How to Run Locally

Since this is a purely static webpage built with vanilla web technologies, there are no build steps or heavy node modules needed!

1. Simply open the folder containing these files.
2. Double-click the `index.html` file to instantly open and interact with the website in your web browser.
